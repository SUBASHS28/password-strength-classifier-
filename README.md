## Project Description
This project is a web-based Password Strength Classifier and Secure Password Generator.
It evaluates password strength using predefined security rules and provides real-time
visual feedback. The system also generates strong random passwords to promote
cybersecurity best practices.

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- Real-time password strength analysis
- Visual strength indicator
- Secure password generator
- Client-side execution (no data storage)
